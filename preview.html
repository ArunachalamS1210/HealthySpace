Get Started with the Full Dashboard

Quick Setup (Windows)

  1. 1. Install Python 3.8+ from python.org
  2. 2. Double-click setup.bat to install dependencies
  3. 3. Double-click run.bat to start the dashboard
  4. 4. Open http://localhost:5000 in your browser

Manual Setup

  1. 1. Create virtual environment: python -m venv venv
  2. 2. Activate: venv\Scripts\activate
  3. 3. Install deps: pip install -r requirements.txt
  4. 4. Run app: python app.py
